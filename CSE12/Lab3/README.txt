Lab 3: Looping in RISC-V Assembly
-----
Description: 
Program prints and creates a file containing a drawing of a right angle triangle using stars and spaces;
the size of which depends upon the integer input of the user. For example, if the user input was to be 
six, the program would draw a six star tall right angle triangle with a hollow interior.
--------

Lab3.asm

Instructions:
This program is intended to be run using the RISCV Assembler and Runtime Simulator
(RARS). Enter the test case as a program argument and run using RARS
